# GenAI_powered_meeting_assistant
Using Whisper AI model to build a meeting summarizer by transforming speech to text

In this project, I used OpenAI's Whisper to transform speech to text. 
Then,I used IBM Watson's AI to summarize and find key points. 
I made make an app with Hugging Face Gradio as the user interface.


I created different LLM applications with Gradio interface
